# wplog

A water polo secondary game log — a progressive web app (PWA) for logging game events poolside and producing official-looking game sheets.

## Features

- **Live event logging** — minimal-tap interface optimized for poolside use
- **Official game sheet** — renders a clean, printable "Progress of Game" log
- **Offline-first** — works without internet connectivity (PWA)
- **Multi-format support** — USAWP, NFHS (High School), and NCAA rule sets
- **QR sharing** — share game data via compressed QR code
- **No backend required** — all data stays on-device in localStorage

## Getting Started

Serve the app locally with any static file server:

```sh
python -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
