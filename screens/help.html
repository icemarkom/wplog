Quick Reference

Quick Start

  1. Choose your Rules from the dropdown
  2. Fill in game details (date, time, location, teams)
  3. Choose logging mode in Logging Mode (Game Log, Stats, or both)
  4. Tap Start Game
  5. Log events from the Live screen
  6. View the game sheet in Sheet; export CSV or download game data from Share

Logging Events

  1. Tap an event button (Goal, Exclusion, etc.)
  2. Enter time — digits fill right-to-left as M:SS (MM:SS for periods 10m+)
  3. Enter cap number
  4. Select team (White or Dark)
  5. Tap OK

Logging Stats

  • Stats buttons appear in teal below log event buttons
  • Time entry for stats depends on your Logging Mode settings
  • Stat events do not affect the score or foul tracking

Cap Numbers

  • Digits 1-99 for field players
  • Goalies: 1A, 1B, 1C (tap 1 then A/B/C)
  • Cards: C for coach, AC for assistant, B for bench
  • Swap Caps: Use the grey Swap Caps button to swap or replace a cap to track a player's true identity. Time is optional.

Player Names

  • A Name field appears below the team buttons when logging player events
  • Names are optional — leave blank to log cap only
  • Once entered, the name auto-fills the next time you log against the same cap
  • Use the grey Add Name button to add player names without logging an event
  • On the Sheet tab, tap any roster row to edit a player's name or ID inline
  • The Roster section at the bottom of the game sheet lists all players with their names
  • Use the Roster toggle in the Print dialog to include or exclude the roster from printouts
  • Some rule sets show an additional ID field (e.g., Membership ID)

Roster Upload & Download

  • Tap the ↑ button next to a team name on Setup to upload a CSV roster
  • CSV format: header row with Cap,Name columns (optional ID column)
  • Upload merges into the existing roster — matching caps are overwritten
  • Duplicate caps in the file are warned and skipped (first entry wins)
  • Upload is available before and during an active game
  • On the Share tab, tap Download Rosters to export as CSV
  • Choose White, Dark, or Both teams to download

Periods

  • End Period advances to the next period
  • End Game appears when the score is not tied
  • Delete an End of Period entry to revert

Overtime & Shootout

  • Enable OT or SO in Setup (mutually exclusive)
  • OT plays in pairs (OT1 → OT2); unlimited if still tied
  • Shootout locks time to 0:00

Timeouts

  • TOL shows timeouts remaining per team
  • Warning when a team exceeds their limit

Foul Tracking

  • Personal fouls accumulate toward the foul-out limit
  • Some events trigger immediate removal from the game (MC, BR, E-Game, FM)
  • A popup alerts you when a player fouls out

Game Sheet & Export

  • Sheet tab shows a printable game summary
  • Tap CUMULATIVE / PER PERIOD on the sheet to toggle stat modes
  • Share tab → Print Game Sheet to precisely select and print the Log, Stats, or Both
  • Download CSV or Download Game Data to export digital files

Save & Load

  • Download Game Data on the Share tab saves the game as a JSON file
  • Load Game on the Setup tab imports a previously saved game file
  • Loaded games restore all events, scores, and settings

Tips

  • The screen should automatically stay awake when tracking an active game on the Live tab
  • Tap ⬌ on Setup to swap Home/Away team colors
  • Tap a filled time or cap field to clear and re-enter
  • Tap any event in the game log to edit its time, cap, team, or period — the event type cannot be changed (delete and re-add instead)
  • Missed event? Tap any event button, then select a previous period in the pill selector to log it in the correct period
  • Delete any event from the game log with the ✕ button
  • Setup fields (date, teams, etc.) stay editable during a game
  • On desktop, use the keyboard in the event dialog: digits for time/cap, A/B/C for letters, W/D for team, O for official, Tab to switch fields, Enter to confirm, Esc to cancel
