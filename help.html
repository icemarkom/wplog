Quick Reference

Quick Start

  1. Choose your Rules (USAWP, NFHS Varsity, NFHS JV)
  2. Fill in game details (date, time, location, teams)
  3. Tap Start Game
  4. Log events from the Live screen
  5. View the game sheet in Sheet, print from Share

Logging Events

  1. Tap an event button (Goal, Exclusion, etc.)
  2. Enter time — digits fill right-to-left as M:SS
  3. Enter cap number
  4. Select team (White or Dark)
  5. Tap OK

Cap Numbers

  • Digits 1–99 for field players
  • Goalies: 1A, 1B, 1C (tap 1 then A/B/C)
  • Cards: C for coach, AC for assistant coach

Periods

  • End Period advances to the next period
  • End Game appears when the score is not tied
  • Delete an End of Period entry to revert

Overtime & Shootout

  • Enable OT or SO in Setup (mutually exclusive)
  • OT plays in pairs (OT1 → OT2); unlimited if still tied
  • Shootout locks time to 0:00

Timeouts

  • TOL shows timeouts remaining per team
  • Warning when a team exceeds their limit

Foul Tracking

  • Personal fouls accumulate toward the foul-out limit
  • Some events trigger immediate removal from the game (MC, BR, E-Game, FM)
  • A popup alerts you when a player fouls out

Game Sheet & Printing

  • Sheet tab shows a printable 2-page game summary
  • Share tab → Print Game Sheet to print or save as PDF

Tips

  • Tap a filled time or cap field to clear and re-enter
  • Delete any event from the recent log with the ✕ button
  • Setup fields (date, teams, etc.) stay editable during a game
