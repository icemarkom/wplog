Quick Reference

Quick Start

  1. Choose your Rules from the dropdown
  2. Fill in game details (date, time, location, teams)
  3. Choose logging mode in Logging Mode (Game Log, Stats, or both)
  4. Tap Start Game
  5. Log events from the Live screen
  6. View the game sheet in Sheet; print, export CSV, or download game data from Share

Logging Events

  1. Tap an event button (Goal, Exclusion, etc.)
  2. Enter time — digits fill right-to-left as M:SS
  3. Enter cap number
  4. Select team (White or Dark)
  5. Tap OK

Logging Stats

  • Stats buttons appear in teal below log event buttons
  • Time entry for stats depends on your Logging Mode settings
  • Stat events do not affect the score or foul tracking

Cap Numbers

  • Digits 1–99 for field players
  • Goalies: 1A, 1B, 1C (tap 1 then A/B/C)
  • Cards: C for coach, AC for assistant, B for bench

Periods

  • End Period advances to the next period
  • End Game appears when the score is not tied
  • Delete an End of Period entry to revert

Overtime & Shootout

  • Enable OT or SO in Setup (mutually exclusive)
  • OT plays in pairs (OT1 → OT2); unlimited if still tied
  • Shootout locks time to 0:00

Timeouts

  • TOL shows timeouts remaining per team
  • Warning when a team exceeds their limit

Foul Tracking

  • Personal fouls accumulate toward the foul-out limit
  • Some events trigger immediate removal from the game (MC, BR, E-Game, FM)
  • A popup alerts you when a player fouls out

Game Sheet & Printing

  • Sheet tab shows a printable game summary
  • When stats are enabled, Player Stats tables show per-team with a row per player and a column per stat
  • Share tab → Print Game Sheet to print or save as PDF
  • Print dialog offers Totals Only or Per Period stats detail when stats are enabled

Save & Load

  • Download Game Data on the Share tab saves the game as a JSON file
  • Load Game on the Setup tab imports a previously saved game file
  • Loaded games restore all events, scores, and settings

Tips

  • Tap a filled time or cap field to clear and re-enter
  • Delete any event from the recent log with the ✕ button
  • Setup fields (date, teams, etc.) stay editable during a game
  • On desktop, use the keyboard in the event dialog: digits for time/cap, A/B/C for letters, W/D for team, O for official, Tab to switch fields, Enter to confirm, Esc to cancel
